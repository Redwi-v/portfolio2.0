# Andrey Kaysin - Portfolio Website

Уникальный, современный портфолио-сайт

## 🎨 Основные особенности дизайна

### Уникальный дизайн
- **Глитч-эффекты** на заголовках
- **Неоновое свечение** и пульсация
- **Геометрические фигуры** на фоне
- **Частицы** с анимацией
- **3D эффекты** при наведении
- **Кастомные формы** (шестиугольники, ромбы)
- **Креативные переходы** между секциями

### 🌐 Мультиязычность
- **Русский** и **Английский** языки
- Переключатель языка в шапке
- Весь контент доступен на обоих языках

### 🎭 Темная/Светлая тема
- **Темная тема** (по умолчанию) - тёмно-синий фон (#1a1a2e) с неоновым зелёным акцентом (#00ff88)
- **Светлая тема** - чистый белый дизайн с теми же акцентами
- Плавные переходы между темами
- Сохранение выбранной темы

### ✨ Креативные анимации
- **Загрузочный экран** с анимированным логотипом
- **Глитч-эффект** на главном заголовке
- **Пульсирующее свечение** на элементах
- **Вращающиеся частицы** вокруг аватара
- **Анимация прогресс-баров** навыков
- **3D наклон карточек** при наведении
- **Анимированные счётчики** статистики
- **Плавающие иконки** кода
- **Морфинг форм** и переходы

### 📱 Адаптивность
- Мобильная версия с гамбургер-меню
- Оптимизация для всех размеров экранов
- Touch-friendly интерфейс

   - Геометрические фигуры на фоне

   - Шестиугольные соц. иконки

   - Анимированный логотип с вращением

   - Концентрические круги на фоне

   - Огромные номера на фоне
   - Вращающиеся рамки иконок

4. **Stats**

6. **Blog**

7. **Testimonials**
   - Карусель отзывов
   - Рейтинг звёздами

8. **Contact**
   - Контактные карточки с цветами
   - Форма с неоновым фокусом

9. **Footer**
   - Quick Links
   - Социальные сети
   - Кнопка "Наверх"
   - Декоративные углы

## 🚀 Технологии

- **React 18** - библиотека UI
- **Tailwind CSS v4** - utility-first CSS
- **Motion** (motion/react) - продвинутые анимации
- **next-themes** - управление темой
- **Lucide React** - иконки
- **TypeScript** - типизация

## 🎨 Цветовая схема

### Темная тема (по умолчанию)
```css
--background: #1a1a2e
--card: #252540
--neon-green: #00ff88
--neon-green-dark: #00cc6a
```

### Светлая тема
```css
--background: #ffffff
--card: #ffffff
--neon-green: #00ff88 (тот же акцент)
```

## 🎭 Уникальные фишки

1. **Глитч-эффект** - случайное искажение текста каждые 3 секунды
2. **Particle System** - Canvas-анимация частиц с линиями связи
3. **Hexagonal shapes** - шестиугольные формы для иконок
4. **3D Card tilt** - карточки наклоняются при hover
5. **Neon glow pulse** - пульсирующее неоновое свечение
6. **Morphing animations** - плавные морфинг-переходы
7. **Floating elements** - плавающие декоративные элементы
8. **Custom clip-paths** - уникальные формы через clip-path
9. **Gradient overlays** - многослойные градиенты
10. **Animated counters** - счётчики с плавной анимацией

## 📁 Структура проекта

```
src/
├── app/
│   ├── components/
│   │   ├── Header.tsx           # Навигация с переключателями
│   │   ├── Hero.tsx             # Главная секция с глитчем
│   │   ├── About.tsx            # О себе + навыки
│   │   ├── Services.tsx         # 3D карточки услуг
│   │   ├── Projects.tsx         # Портфолио проектов
│   │   ├── Stats.tsx            # Анимированная статистика
│   │   ├── Blog.tsx             # Блог секция
│   │   ├── Contact.tsx          # Контактная форма
│   │   ├── Testimonials.tsx     # Карусель отзывов
│   │   ├── Footer.tsx           # Подвал
│   │   ├── ScrollToTop.tsx      # Кнопка наверх
│   │   ├── PageLoader.tsx       # Загрузочный экран
│   │   └── ParticleBackground.tsx # Частицы
│   ├── contexts/
│   │   └── LanguageContext.tsx  # Контекст языка
│   ├── hooks/
│   │   └── useInView.ts         # Intersection Observer
│   ├── translations.ts          # RU/EN переводы
│   └── App.tsx                  # Главный компонент
├── styles/
│   ├── theme.css                # Темы + кастомные утилиты
│   └── fonts.css                # Google Fonts
```

## 🎯 Производительность

- Intersection Observer для lazy-анимаций
- CSS variables для быстрой смены темы
- Оптимизированные re-renders
- Canvas для частиц (GPU acceleration)
- Motion animations с spring physics
